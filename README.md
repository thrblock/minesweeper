# minesweeper
经典扫雷复刻!

基于之前开发的cino项目搭建
[GitHub](https://github.com/thrblock/cino)

## 🚩回顾一下具体规则
![F1](https://raw.githubusercontent.com/thrblock/minesweeper/master/mineSrc/1.png) 
* 翻开所有非雷格子获胜
* 鼠标左键点击可以翻开一个格子
* 格子上的数字代表周围8格范围内雷的数量
* 用鼠标右键将一个格子标记为地雷，再次右键取消标记
* 翻开地雷格子则失败
